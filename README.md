## LampaParty - Watch with friends

> Модуль базується на [WatchTogether](https://github.com/lampac-nextgen/lampac/tree/main/Modules/WatchTogether) із проєкту `lampac-nextgen/lampac`.

Модуль для **lampac NextGen**. Спільний перегляд у Lampa: створюєте кімнату, ділитеся посиланням з друзями - і всі дивляться один і той самий потік синхронно.

### Що робить

- Кнопка в шапці Lampa відкриває **браузер кімнат** зі списком відкритих кімнат і замочком для запаролених.
- Створення кімнати:
  - **за прямим посиланням** (m3u8 / mp4);
  - **із поточного потоку** (хост уже щось дивиться);
  - **з контекстного меню потоку** (`hover:long` на серії) або **з налаштувань плеєра** (шестерня) - пункт **«LampaParty - Дивитися з друзями»**.
- Синхронізація плеєра по WebSocket: play / pause / seek, м'яка корекція темпу при невеликій розбіжності та жорсткий seek при сильній.
- Сповіщення про дії учасників з іменем ініціатора (вхід, вихід, пауза, відновлення, перемотування).
- **Передача хоста**: якщо хост вийшов, а в кімнаті лишилися учасники - роль автоматично переходить до наступного, кімната живе далі.
- **Переключення серій хостом** - усі учасники автоматично переходять на нове джерело без перепідключення.
- Захист паролем (опційно), локалі `uk` / `en` / `ru`, бейдж кімнати з лічильником глядачів поверх плеєра.
- Стійкість до проксі з idle-timeout (Cloudflare): двосторонній ping/pong + серверний heartbeat кожні 20 с.

### Як підключити через `repository.yaml`

У файлі `module/repository.yaml` додайте запис:

```yaml
- repository: https://github.com/nrsua/lampac-nrs
  branch: main
  modules:
    - LampaParty
```

Lampac автоматично завантажить модуль і скомпілює його на льоту (`dynamic: true` - без бінарників).

#### Налаштування в Lampa

URL плагіна: `http(s)://<хост>:<порт>/lampaparty.js`

Після перезапуску в шапці Lampa з'явиться іконка **LampaParty**, а в меню серій / у налаштуваннях плеєра - пункт **«LampaParty - Дивитися з друзями»**.

У `Lampa.Settings` під пунктом **LampaParty**:

- **Ім'я користувача** - порожньо означає `lampac_unic_id`.
- **Використовувати пароль** - чи підставляти поле пароля при створенні.
- **Пароль за замовчуванням** - підказка для входу / створення.

### Вимоги

- Працюючий **NWS** (`Startup.Nws`) на стороні lampac.
- Стандартні `Shared` і `Core`.

## EpWatch - Episode Watcher

Модуль для **lampac NextGen**. Стежить за новими серіями ваших улюблених серіалів і надсилає сповіщення в Telegram.

### Що робить

- Підписка прямо з картки серіалу в Lampa (кнопка **EpWatch**).
- Вибір **конкретної озвучки** з усіх онлайн-балансирів, які налаштовані у вашому lampac (UAFlix, Filmix і будь-які кастомні - модуль опитує їх через стандартний `/lite/events`).
- Вибір **конкретного сезону** після озвучення, включно з майбутніми (наприклад, підписатися на S10, який ще не вийшов).
- Telegram-бот:
  - `/list` - список підписок з прогрес-барами `▓░░░░░░░░░` і статусами 🟢/🔵/🟡/✅/🔴.
  - `/check` - миттєва перевірка з підсумком «що нового + повний статус».
  - `/lang` - мова інтерфейсу (`uk` / `en` / `ru`), окрема для кожного користувача.
  - `/unlink` - відв'язати акаунт.
- Фоновий чекер раз на годину опитує тільки потрібні балансери (з троттлінгом, паузою до дати наступної серії з TMDB, кешем).
- Окрема сторінка **EpWatch** у меню Lampa зі списком підписаних серіалів.
- HTML-форматування повідомлень: `<blockquote>`, постер серії як preview, expandable-опис, прогрес-бари сезону.

### Як підключити через `repository.yaml`

У файлі `module/repository.yaml` додайте запис:

```yaml
- repository: https://github.com/nrsua/lampac-nrs
  branch: main
  modules:
    - EpWatch
```

Lampac автоматично завантажить модуль, скомпілює (модуль `dynamic: true` - компілюється на льоту, без бінарників) і запустить TelegramBot + чекер.

#### Налаштування у `init.conf`

Додайте секцію:

```jsonc
"EpWatch": {
  "enable": true,
  "bot_token": "ТОКЕН_БОТА"
}
```

Мінімум, що потрібно - `bot_token`.

#### Встановлення плагіна в Lampa

URL: `http(s)://<хост>:<порт>/epwatch.js`

Після перезапуску у меню Lampa з'явиться пункт **EpWatch**, а в картках серіалів - кнопка **EpWatch**.
